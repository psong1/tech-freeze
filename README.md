## Introduction

Our project is a note taking app. You will be able to create a login and password on the home page. After you create this unique to you login, you will be able to write and save notes with this application.

## Installation

Click the repository link below. Download the zip file and copy it to a directory of your choice or clone to a directory of your choice using your terminal. Open the file in VS Code then open the terminal and make sure you’re in the root directory 'TECH-FREEZE'. After installation is complete, run the following code: ‘node inex.js’ in the terminal. You’ll now be ready to run the app!

## Usage
- After accessing the link into your desired browser, you will see the login/sign up page. 
-  After accessing the link into your desired browser, you will see the login/sign up page. 
-  You can either enter your login credentials if you have visited our site before OR click on the ‘Signup’ button to go to to the next page.
- If you already have an account with our app, after you login, you will be directed to your notes.
- If you do not have an account and clicked on ’Signup’ then, you will be prompted to enter in a username of your choice, the email you will use, and a password of your choosing. 
- Click ’Submit’ to be directed back to the login page.
- From the login page, you will now enter in your chosen email and password that you typed in on the previous signup page and click ’Submit’.
-  You will be directed to the main note taking page. 
- On this page, you can add notes, simply by clicking on the blue ‘Add’ on the left side.
- After clicking on ‘Add’, a blank note template will show on your screen.
- When you click on the heading ‘Untitled Note’ you can change the name of your note.
- Underneath, there is a white box with the text ‘Write your note here…’ This is where you click on that space and start typing whatever you’d like to add.
- The note you made will save automatically and you are able to find it on the left hand side of your screen in a list.
- When you edit a note from anywhere on your list, it will be pushed to the top of the list. 
- The date and time for new or edited notes will be displayed on the left underneath the note.
